🌳 家系図作成ツール / Pedigree Creation Tool

Serverless / GitHub Pages - Powered by draw.io

🚀 ツールを使う / Use the Tool

PEDファイル形式で家系図を作成できます。データはブラウザ内のみで処理され、サーバーには送信されません。

Create pedigree diagrams from PED format. All data is processed locally in your browser.

家系図作成ツールを開く / Open Tool
※ データはWebサーバーに保存されません。作成した家系図はPNGまたはSVGで保存してください。
※ Data is NOT saved on the server. Please save your pedigree as PNG or SVG.

🪄使い方 / How to Use

① PEDファイルを準備してください

カラム順: 1.家系ID 2.SampleID 3.父SampleID 4.母SampleID 5.性別 6.表現型 7.年齢

  • 1.家系ID
  • 2.SampleID
  • 3.父SampleID : 不明の場合は 0 をご指定ください
  • 4.母SampleID : 不明の場合は 0 をご指定ください
  • 5.性別 : 男性＝1、女性＝2 と記載してください。空白の場合は、性別不明=ダイヤで表示します
  • 6.表現型 : 未発症＝1、発症＝2 と記載してください。空白の場合は未発症扱いになります
  • 7.年齢 : 空白としても問題ありません。兄弟は年齢順にソートします

（例）

FID	ID	父ID	母ID	性別	表現型	年齢
FAM002	I001	0	0	1	1	42
FAM002	I002	0	0	2	1	40
FAM002	I003	I001	I002	1	1	14
FAM002	I004	I001	I002	2	2	10

② PEDファイル貼り付け枠にコピー＆ペーストしてください

8カラム目以降は無視されます。

③「貼り付け内容で置き換える」ボタンを押します

その下にテーブルが出現します。テーブルに表示されている内容が図になるので確認してください。

④「PEDから家系図生成して開く」ボタンを押します

draw.io の描画画面が開きます。

⑤ draw.io で記号の追加や形の調整を行います

〇や□や線の記号は図形ライブラリに追加しています。

⑥「PNGで保存する」または「SVGとして保存する」ボタンを押します

画像ファイルがダウンロードされますので、保存してください。

⑦ パワーポイントで追加編集する場合

SVGで保存して、パワーポイントファイルにSVG画像を挿入します。

  • パワーポイント上で画像を右クリックして「図形に変換」ボタンを押すと、パワーポイントの図形オブジェクトとして編集できます。
  • 「グループ解除」をするとオブジェクトが一つずつに分かれます。
