🌳 家系図作成ツール（クライアント完結 / 静的配信）

GitHub Pages上で、同一ページ内に draw.io を読み込んで動作します

データはWebブラウザ内で処理され、外部サーバーには送信されません。このため編集中のファイルは保存できません。

📋 PED入力（Draw.io起動前）

【続柄TSVを入力してPEDへ変換】
！1家系ずつ登録してください。家系IDは最初の行の値を使用します。
カラムは、プロジェクト指定の列を入力してください: 家系ID / 統合ID / 発端者／続柄 / 性別 / 発症状況 / 生年月
カラム名（ヘッダ）も同時にペーストしてください。カラムの並び順は問いません。
この欄の内容は内部でPEDに変換され、下のPEDテーブルへ反映されます。
必要に応じて家系図作成のためのダミー親を追加作成します。
【一例ずつ入力する場合は以下に入力して、「一人追加」ボタンを押してください。テーブルに反映されます】
【PEDファイル（タブ区切り）をお持ちの場合は下の枠内に貼り付けてください】
カラム順: 1.家系ID 2.SampleID 3.父SampleID 4.母SampleID 5.性別 6.表現型 7.年齢
1.家系ID
2.SampleID
3.父SampleID : 不明の場合は 0 をご指定ください
4.母SampleID : 不明の場合は 0 をご指定ください
5.性別 : 男性＝1、女性＝2 と記載してください。空白の場合は、性別不明=ダイヤで表示します
6.表現型 : 未発症＝1、発症＝2 と記載してください。空白の場合は未発症扱いになります
7.年齢 : 空白としても問題ありません。兄弟は年齢順にソートします
📋 以下のテーブルの内容で家系図を作成します。確認してください
